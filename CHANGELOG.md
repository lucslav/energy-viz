# Changelog

## [1.0.0] - 2026-03-12
### Added
- **Smart File Detection:** Automatic recognition of 4 ESB file types (kWh, kW, DNP, Cumulative).
- **Power Demand Mode:** High-visibility line charts for kW spikes with interactive range sliders.
- **Unified UI:** Fully English interface with professional "Timestamp" axis labels.
- **24h Tariff Label:** Improved clarity for basic daily total files.
- **Impersonal UX:** Updated all guides and recommendations to a formal, technical tone.

### Fixed
- Range slider glitch in Power Demand mode (mini-graph visibility).
- VAT calculation consistency across different tariff modes.
